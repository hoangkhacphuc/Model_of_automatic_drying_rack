### Chú thích
- `#define Led wPi` : Trong đó ***wPi*** là giá trị wPi tương ứng với chân vật lý cắm Led.
- `#define Button wPi` : Trong đó ***wPi*** là giá trị wPi tương ứng với chân vật lý cắm Button.
